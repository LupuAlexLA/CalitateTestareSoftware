Tema propusa este de a vedea kilometrajul masinii personale, de fiecare data cand parcurg o anumita distanta.

In acest scop voi avea in vedere 3 chestiuni:
- data la care am observat
- kilometrajul initial - kilometrii inainte de a incepe parcurgerea distantei
- kilometrajul final - kilometrii dupa parcurgerea distantei

Am decis sa salvez aceste informatii intr-un fisier csv, adaugand doua coloane:
- id
- kilometrii parcursi ( diferenta dintre kilometrajul initial si cel final )

Comunicarea dintre client si server se realizeaza prin socket de tip udp, single thread.

Clientul are trei actiuni posibile, pe care le va specifica in consola:
- GET ( la accesarea acestei comenzi, serverul va returna o lista de kilometraje )
- INSERT ( insereaza un kilometraj in fisierul csv )
- PUT ( updateaza un kilometraj in fisierul csv, in functie de un id )

Serverul va scrie in consola tipul actiunii primite de la client
